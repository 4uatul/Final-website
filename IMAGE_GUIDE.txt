═══════════════════════════════════════════════════════════════════
                    IMAGE REQUIREMENTS
═══════════════════════════════════════════════════════════════════

📸 PROFILE PICTURE (profile.jpg)
   ├─ Format: JPG or PNG
   ├─ Recommended Size: 500x500px or larger
   ├─ Shape: SQUARE (will be displayed as circular)
   ├─ Quality: High resolution, professional headshot
   └─ Tips: 
      • Good lighting
      • Professional attire
      • Plain or blurred background
      • Smile! 😊


🖼️  PROJECT IMAGES (project1.jpg, project2.jpg, project3.jpg)
   ├─ Format: JPG or PNG
   ├─ Recommended Size: 800x500px (16:10 ratio)
   ├─ Shape: RECTANGULAR (landscape orientation)
   ├─ Quality: High resolution, clear screenshots
   └─ Tips:
      • Use actual project screenshots
      • Show the UI/interface
      • Include data visualizations if possible
      • Clean, uncluttered view


═══════════════════════════════════════════════════════════════════
                WHERE TO GET PROJECT IMAGES
═══════════════════════════════════════════════════════════════════

Option 1: TAKE SCREENSHOTS of your projects
   • Open your project in browser/application
   • Press Windows Key + Shift + S (Windows)
   • Or Cmd + Shift + 4 (Mac)
   • Capture the most impressive view
   • Crop to 800x500px

Option 2: CREATE MOCKUP IMAGES
   • Use Canva.com (free)
   • Use Figma (free)
   • Show data visualizations
   • Show architecture diagrams

Option 3: USE PLACEHOLDER IMAGES (Temporary)
   • The website already has placeholders built-in
   • They'll show colored boxes until you add real images
   • Replace them when you have real screenshots

═══════════════════════════════════════════════════════════════════
                    NAMING YOUR FILES
═══════════════════════════════════════════════════════════════════

✅ CORRECT FILE NAMES:
   ├─ profile.jpg
   ├─ project1.jpg
   ├─ project2.jpg
   └─ project3.jpg

❌ WRONG FILE NAMES (These won't work!):
   ├─ Profile.jpg          (Capital P - case sensitive!)
   ├─ profile.jpeg         (must be .jpg, not .jpeg)
   ├─ profile picture.jpg  (no spaces allowed)
   ├─ project_1.jpg        (use project1, not project_1)
   └─ myproject.jpg        (must match exact names above)

═══════════════════════════════════════════════════════════════════
                    TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

🔧 Image not showing?
   → Check the file name matches EXACTLY (case-sensitive!)
   → Make sure it's in the same folder as portfolio.html
   → Try refreshing your browser (Ctrl + F5 or Cmd + Shift + R)

🔧 Image looks stretched or distorted?
   → Use recommended sizes: 500x500px for profile, 800x500px for projects
   → Make sure profile is square and projects are rectangular

🔧 File size too large?
   → Compress images at: tinypng.com or compressor.io
   → Keep images under 500KB for fast loading
